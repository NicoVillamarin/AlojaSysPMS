# AlojaSys – Tarifas Comerciales (Borrador para Clientes)

> Todos los valores son **de referencia** y están expresados en **pesos argentinos (ARS)**.  
> Para presupuestos formales se recalcula según el contexto de cada cliente.

---

## 1. Resumen rápido por plan

| Plan     | Base por hotel (ARS) | Precio por habitación (ARS) |
|----------|----------------------:|----------------------------:|
| Starter  | 25.000 / mes          | 2.000 / hab / mes           |
| Pro      | 45.000 / mes          | 3.000 / hab / mes           |
| Premium  | 70.000 / mes          | 4.000 / hab / mes           |
| Custom\* | 20.000 / mes          | 2.000 / hab / mes           |

\*En **Custom** se agregan módulos opcionales (AFIP, OTAs, WhatsApp, etc.) con recargos específicos.

---

## 2. Plan Starter – Operación diaria simple

**Pensado para:** hostels y hoteles pequeños que necesitan gestionar habitaciones y reservas sin AFIP, pagos online ni OTAs.

### 2.1 Precio

- **Base por hotel:** **ARS 25.000 / mes**  
- **Por habitación:** **ARS 2.000 / habitación / mes**

- **Fórmula aproximada:**  
  - **Total mensual ≈ ARS 25.000 + (habitaciones × ARS 2.000)**

### 2.2 Ejemplos de referencia

| Habitaciones | Total aprox. (ARS)                  |
|--------------|--------------------------------------|
| 5            | 25.000 + 5×2.000 = **35.000 / mes**  |
| 10           | 25.000 + 10×2.000 = **45.000 / mes** |
| 20           | 25.000 + 20×2.000 = **65.000 / mes** |

> Los rangos de habitaciones (ej. "hasta 20 hab") son **orientativos**.  
> El plan se elige por funcionalidades; el precio escala de forma transparente por habitación.

## 5. Plan Custom – A la carta por módulos

**Pensado para:** proyectos especiales o clientes que quieren armar un paquete específico (ej. solo Channel Manager + WhatsApp, sin AFIP).

### 5.1 Precio base

- **Base por hotel:** **ARS 20.000 / mes**  
- **Por habitación:** **ARS 2.000 / habitación / mes**

### 5.2 Módulos opcionales (referencia)

Valores aproximados **adicionales por hotel / mes**:

| Módulo                               | Recargo (ARS aprox.) |
|--------------------------------------|----------------------:|
| AFIP / Facturación electrónica       | +10.000              |
| Mercado Pago / Pagos avanzados       | +10.000              |
| OTAs / Channel Manager               | +20.000              |
| Chatbot WhatsApp (BYON)              | +5.000               |
| Chatbot WhatsApp (número admin.)     | +10.000–15.000       |
| Housekeeping avanzado                | +10.000              |
| Conciliación / Cobros avanzados      | +10.000              |

### 5.3 Ejemplo de armado Custom

Hotel de **10 habitaciones** que quiere:
- Starter/Custom core + AFIP + WhatsApp BYON.

Cálculo de referencia:

- Base: 20.000  
- Habitaciones: 10 × 2.000 = 20.000  
- AFIP: +10.000  
- WhatsApp BYON: +5.000  
- **Total aprox.: 55.000 ARS / mes**.

---

## 6. Notas finales

- Los precios aquí indicados son **de referencia** y pueden ajustarse según el contexto de cada cliente.  
- Los planes se eligen principalmente por **funcionalidades** (módulos incluidos).  
- La cantidad de habitaciones sólo ajusta el total mediante la parte variable por habitación.  
- Para propuestas formales se recomienda:
  - Indicar el **plan** elegido (Starter, Pro, Premium o Custom).  
  - Detallar **número de habitaciones**.  
  - Enumerar los **módulos adicionales** incluidos (si los hay).  
  - Aclarar si el valor está sujeto a actualización por inflación o por cambios significativos de costos.
